# User Journeys

## Brand Journey

### Onboarding
Sign up → Company profile setup →
Wallet funding → Billing configuration

### Campaign Setup
Build product catalog in CMS →
Create product-specific forms →
Launch campaign wizard →
Set type (CPC/CPV/CPP/CPCall), budget, audience →
Affiliate links auto-generated per campaign

### Discovery & Collaboration
Browse AI-matched creator recommendations →
Filter by niche, engagement, audience size →
Send collaboration invite →
Creator accepts → Links activated

### Monitoring & Payout
Track real-time dashboard (clicks, conversions, ROI) →
Fraud alerts surface anomalous patterns →
Campaign ends →
Automated payout triggered to creator wallets →
Download invoices and reports

---

## Creator Journey

### Onboarding
Sign up → OAuth social verification →
Profile setup (niche, handles, audience size)

### Campaign Execution
Browse marketplace →
Filter by niche, brand, payout type →
Accept campaign → Receive affiliate link →
Share via storefront or social channels →
Track clicks and conversions in dashboard

### Monetization
Build storefront → Add brand products →
Link storefront to social media bio →
Earn from campaigns + storefront sales →
Create courses or subscription plans →
Manage community via integrated platform →
View unified earnings dashboard →
Request withdrawal after hold period clears

---

## Platform Settlement Flow

Campaign ends →
Conversion data reconciled →
Hold period starts →
Brand window to flag disputes →
Hold period clears →
Automated payout to creator →
Transaction logged with downloadable receipt
