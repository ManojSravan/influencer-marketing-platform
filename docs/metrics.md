# Metrics & Impact

> All figures anonymized and directional.

## North Star Metric
Campaign conversion rate (affiliate click → purchase / lead)

## Brand-Side
| Metric | Outcome |
|---|---|
| Campaign setup time | Reduced significantly vs manual outreach |
| Fraud exposure | Anomalous clicks flagged before payouts processed |
| Concurrent campaign management | Brands managing multiple campaigns from one dashboard |
| CMS setup time | Bulk upload reduced catalog setup from days to hours |

## Creator-Side
| Metric | Outcome |
|---|---|
| Revenue streams per creator | Creators consolidated 3+ streams in one platform |
| Storefront activation | Increased affiliate link sharing rate via social bio |
| Recurring revenue | Course and subscription features added predictable income |
| Payout trust | Hold period with real-time pending view reduced disputes |

## Platform-Level
| Metric | Outcome |
|---|---|
| Community feature delivery | Shipped 2x faster via API vs in-house estimate |
| Payout dispute rate | Reduced materially with hold period logic |
| Creator retention | Storefront ownership increased platform stickiness |
